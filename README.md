# TMS — Une loi, tracée dans l'espace

Livre interactif présentant la théorie TMS (Tenseurs Multi-Scalaires) de **Taleb Mohammed Said** (Université de Laghouat) : géométrie complexe, mécanique quantique, science des matériaux, informatique et neurosciences — avec démonstrations mathématiques, simulations réelles et code exécutable.

Site statique, une seule page (`index.html`), sans dépendance de build.

## Développement local

Ouvrir simplement `index.html` dans un navigateur.

## Déploiement

Déployé sur [Vercel](https://vercel.com) comme site statique.
